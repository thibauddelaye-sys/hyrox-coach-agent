# Skills

Reusable instructions for AI assistants working on specific areas of this repo. Each skill is a focused playbook that prevents the assistant from re-deriving the same patterns from scratch.

| Skill | Use when |
|---|---|
| `n8n-workflow-builder/` | Building or modifying any n8n workflow JSON |
| `rag-ingestion/` | Working on the Pinecone ingestion or retrieval flow |
| `airtable-operations/` | Reading from or writing to the Airtable base |
| `telegram-channel-conventions/` | Sending messages to athlete or system channels |

Each skill contains a `SKILL.md` that the assistant should read before acting in that area.
